# github-tipbot

> A GitHub App built with [Probot](https://github.com/probot/probot) that A TurtleCoin tipbot for Github

## Setup

```sh
# Install dependencies
npm install

# Run with hot reload
npm run build:watch

# Compile and run
npm run build
npm run start
```
